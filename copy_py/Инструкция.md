# Инструкция по запуску на Windows

## Быстрый старт:

1. **Скачайте** `Translator_Portable_Windows.zip`
2. **Распакуйте** архив (пароль: `translator2024`)
3. **Запустите** `start_translator.bat`

## Если возникают проблемы:

### Проблема: "Не найден Tesseract"
**Решение:** 
- Запустите `install_tesseract.bat` для автоматической установки
- Или установите вручную: https://github.com/UB-Mannheim/tesseract/wiki

### Проблема: Приложение не запускается
**Решение:**
- Попробуйте `start_simple.bat`
- Отключите антивирус на время запуска
- Запустите от имени администратора

### Функциональность:

- `F2` - Распознать текст вокруг курсора
- `F3` - Перевести распознанный текст  
- `F4` - Выход из приложения

### Особенности:
- Полностью portable - не требует установки
- Работает из любой папки
- Не оставляет следов в системе
